# AEEM - Africa Education Empowerment Movement

> Pioneering inclusive, equitable, and quality education across the African continent.

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Kallon-Tech-savvy/AEEM.git
cd AEEM

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the application.

## 🎨 Design System

### Color Palette
- **Gold**: `#D4AF37` (Primary accent)
- **Charcoal**: `#1A1A1A` (Primary text)
- **Light**: `#F8F7F4` (Background)

### Typography
- **Font Family**: Plus Jakarta Sans
- **Font Sizes**: Responsive scaling with Tailwind

## 📄 Pages

| Page | Route | Purpose |
|------|-------|---------|
| Home | `/` | Main landing page with all sections |
| About | `/about` | Mission, vision, values, and story |
| Impact | `/impact` | Metrics, statistics, and success stories |
| Initiatives | `/initiatives` | Strategic programs and frameworks |

## 🛠️ Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Type Safety**: TypeScript

## 📦 Build & Deploy

```bash

npm run build

# Start production server
npm start

# Check for linting issues
npm run lint
```
